# Brain Tumor Detection Using MRI Images

This project implements a deep learning–based approach for detecting brain tumors from MRI images using Convolutional Neural Networks (CNN) and OpenCV preprocessing techniques.

## Tools & Technologies
- Python
- OpenCV
- TensorFlow
- NumPy, Pandas
- Scikit-learn
- Google Colab

## Dataset
Public MRI dataset obtained from Kaggle.

## Methodology
- MRI image preprocessing using OpenCV
- CNN-based classification
- Evaluation using accuracy, sensitivity, confusion matrix, and ROC–AUC

## Platform
Google Colab with GPU acceleration
